
# Installation & initialisation

After cloning the project:

If you use [NODEJS](https://nodejs.org/en):

`npm i`

Or if you use [BUN](https://bun.sh/):

`bun install`


To start the project:

Start development:

NODEJS: `npm run dev`

BUN: `bun dev`

Start build:

NODEJS: `npm run build`

BUN: `bun run build`
